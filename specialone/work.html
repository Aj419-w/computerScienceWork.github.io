What we can offer

We build and operate virtual machines

we can build virtual machines to function for operating systems that run on your host OS. The virtualised OS is commonly referred to as a guest as is managed by software called a Hypervisor .

The hypervisor helps you allocate resources like CPU, Memory and disk space on your “guest”
Most popular hypervisor we can offer:
  1. Hyper V
  2. Virtualbox
  3. VMware
  4. KVM

The term “cloud” usually means thousands of VMs running on specialised hypervisors on someone else’s server farm in a data centre somewhere else.
Virtual machines allow you to become platform agnostic, meaning you are not limited by particular operating system and tools available for it

Networking

Its understanding how devices interact with each other and how data gets to point A to B
A good understanding in networking will make you and expert troubleshooter. There are 2 conceptual models that govern networking: TCP/IP and OSI
TCP/IP is older than OSI and has 4 layers:
  1. Network access
  2. Internet
  3. transport
  4. Application

    5. OSI has been further updated and has 7 layers:
    1. Physical
    2. Data link
    3. Network
    4. transport
    5. Session
    6. Presentation
    7. Application

    Our engineers are comfortable working with both TCP and OSI so you van be sure we can make a fully functioning OS for all your needs.
